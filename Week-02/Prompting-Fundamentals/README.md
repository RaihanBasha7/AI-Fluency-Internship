# Prompting Fundamentals on Real Tasks v2

## Objective

This assignment explores how prompt engineering techniques improve AI responses using a real task from my AI Workflow Audit (FL-01).

## Real Task

Understand Machine Learning concepts and convert them into structured study notes for internship preparation.

## Techniques Applied

- Naive Prompt
- Role Assignment
- Context and Motivation
- Few-shot Examples
- Output Structure
- Step Decomposition

## Deliverables

- Six prompt versions
- Output comparison
- Claude vs ChatGPT comparison
- Reusable prompt template

## Key Learning

Small improvements in prompt design significantly improved the quality, organization, and usefulness of AI-generated responses. Adding context, examples, and structure produced more accurate, complete, and actionable outputs than a simple one-line prompt.
