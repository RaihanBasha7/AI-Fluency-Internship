# Prompt Ladder

This assignment demonstrates how a weak prompt can be improved through prompt engineering by changing one layer at a time.

## Objective

- Start with a weak prompt
- Improve it through five iterations
- Observe how each individual change affects the output
- Produce a reusable final prompt

## Layers Used

1. Clear Goal
2. Audience
3. Context
4. Output Format
5. Quality Criteria

## Files

- `prompt-ladder.md` — Complete assignment
- `screenshots/` — Evidence of every prompt run
